# Северный Инжиниринг — сайт строительной компании

Многостраничный статический сайт с анимациями, 3D-сценой и формами обратной связи.

## 🚀 Быстрый старт
1. Разместите файлы на GitHub Pages (корень репозитория).
2. Настройте данные в `assets/js/config.js`: название, телефон, соцсети, токен Telegram-бота.
3. Добавьте изображения в `assets/img/` согласно списку ниже.
4. Замените `og-image.jpg` на свой.
5. Подключите Яндекс.Метрику или другие счётчики (опционально).

## 📁 Структура
- `index.html` — главная
- `services.html` — услуги
- `projects.html` — проекты
- `about.html` — о компании
- `contacts.html` — контакты
- `privacy.html`, `terms.html` — документы
- `assets/css/style.css` — стили
- `assets/js/config.js` — все настройки
- `assets/js/main.js` — логика сайта
- `assets/img/` — изображения
- `assets/video/` — видео
- `assets/lottie/` — Lottie-анимации

## 🔧 Используемые библиотеки (CDN)
- GSAP (3.12.5)
- Three.js (0.160)
- Lenis (последняя)
- Swiper (9.x)
- Splitting.js (1.0.6)
- Matter.js (для будущих переходов)
- Leaflet (для карты)
